🎲 Sorteador de Números

Um sorteador de números simples, rápido e intuitivo, desenvolvido para gerar resultados de forma prática.

🚀 Funcionalidades

🔢 Geração de números aleatórios com ou sem repetição

🔁 Botão de sortear novamente

🎨 Interface moderna e responsiva

Acesse o projeto online:
👉 https://sorteador-de-numeros-user.vercel.app/


⚙️ Como usar

Defina os valores desejados

Clique em Sortear

Aguarde a animação

Veja o resultado exibido na tela

Clique em Sortear novamente para repetir ou clique em voltar para alterar os valores desejados


🛠️ Tecnologias utilizadas

HTML5

CSS3

JavaScript
